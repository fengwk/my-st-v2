# My Simple Terminal V2

个人st环境

# Installation

```shell
git clone git@github.com:fengwk/my-st-v2.git ~/prog/my-st-v2
cd ~/prog/my-st-v2
sudo make clean install
```

# Usage

- `alt + pageUp` - 上滚10行
- `alt + pageDown` - 下滚10行
- `alt + shift + pageUp` - 上滚一页
- `alt + shift + pageDown` - 下滚一页
- `ctrl + shift + pageUp` - 放大
- `ctrl + shift + pageDown` - 缩小
- `alt + shift + l` - 打开url
- `alt + y` - 复制url
- `alt + ctrl + y` - 打开url
- `alt + o` - 复制命令输出
